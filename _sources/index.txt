Welcome to nginxCtl's documentation!
====================================
This is a simple tool to manage nginx service on a server. This tool is similar to
apachectl but for nginx. The important feature  of this tool is ability to list vhosts configured on a
nginx server.

Usage::

    python nginxCtly.py -h
    Usage: /root/nginxctl/nginxCtl.py [option]
    Example: /root/nginxctl/nginxCtl.py -v


    Available options:
    -S list nginx vhosts
    -t configuration test
    -k start|stop|status|restart
    -v version
    -h help
